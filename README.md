# Octapad

A Flutter-based SPD-30 / Octapad inspired drum pad application.

## Features

- 8 Drum Pads
  - Crash
  - FX
  - Tom1
  - Tom2
  - HiHat1
  - HiHat2
  - Snare
  - Kick

- Multiple Drum Kits
  - Jazz Kit
  - Rock Kit
  - EDM Kit
  - Indian Kit
  - Percussion Kit

- SPD-30 Inspired Layout
- Kit Selection Panel
- Responsive Flutter UI
- Audio Playback using audioplayers

## Tech Stack

- Flutter
- Dart
- audioplayers

flutter run
